# PM Interview 1h Rescue

A Verdent skill for creating a customized last-hour product manager interview rescue pack from a resume and target JD.

## What it does

This skill turns a candidate's resume and target product manager JD into a practical interview battle plan:

- JD hiring-intent decoding
- Resume-JD fit diagnosis
- Candidate positioning
- Top 3 strongest experience stories
- Self-introduction scripts
- High-probability interview questions
- Risk-question defenses
- PM case frameworks by role type
- Interviewer questions
- 60/30/15-minute cram plans

## Usage

Install or copy the skill folder into your Verdent skills directory, then trigger it with requests such as:

> 这是我的简历和产品经理岗位JD，帮我做一个面试前1小时急救包。

> I have a PM interview in 30 minutes. Here is my resume and the JD. Help me prepare.

## Files

- `SKILL.md` — main skill instructions and trigger metadata
- `references/output-template.md` — full rescue-pack output structure
- `references/role-archetypes.md` — PM role archetypes and interview priorities
- `references/question-bank.md` — high-frequency questions and defense patterns
- `references/time-modes.md` — 60/30/15-minute preparation modes
- `pm-interview-1h-rescue.skill` — packaged skill file
